Operators

Using operators in python

Arithmetic Operators

Arithemetic operators are used to perform mathmatical operations like addition, subtraction and multiplication. These actions can be chained to perform more complicated calculations.

#Addition print(12 + 6) #Subtraction print(4 - 2) #Multiplication print(3 * 6) #Division print(10 / 2) #Chaining print(1 + 2 * 3 / 4)

Another important operation is the modulo % operation which returns the remainder of the division.

#Will return 2 print(17 % 5) #Will return 0 print(6 % 2)

Two multiplication symbols ** creates a power operator

#Will return 49 print(7 ** 2) #Will return 8 print(2 ** 3)

Operators on strings

Python supports concatenating strings using the addition operator

#Hello World print("hello" + " " +"world")

It also supports multiplying (repeating) strings using the multiplication operator

#Lots of hello print("hello" * 5)

Python does not support using operations combining different types of variables

#Will Error print("string" + 9)

Casting

There may be times where we need to convert one variable type to another.

  • To convert a variable to an integer use the int() function.

  • To convert a variable to an string use the str() function.

  • To convert a variable to an float point number use the float() function.

#int() a = int(7.5) #will be 7 b = int("4") #will be 4 #str() c = str(2) #will be '2' d = str(3.2) # will be '3.2' #float() e = float(1) #will be 1.0 f = float("4.2") # will be 4.2

Comparison Operators

Comparison Operators are used to compare to values. The most common is the Equal == operator, checking if two values are equivalent.

a = 5 b = 4 #Equal print(a == b) #Not Equal print(a != b) #Greater than print(a > b) #Less than print(a < b) #Greater than or equal to print(a >= b) #Less than or equal to print(a <= b)
