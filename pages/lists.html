Lists

A list is a collection of data, stored under a single variable. In Python they are created with square brackets [ and each element is seperated by a comma ,.

#Example of a list myList = [1,2,3] print(myList)

We can access an index of the list, similar to how we access a variable. The only difference is including an index offset [n] where n is the index of the element. Something to note is that indexes start at 0, so to access the second element use [1].

myList = [1,2,3] print(myList[2]) #Prints the third element of the list

Accessing an element which does not exist produces an error.

myList = [1,2,3] print(myList[5]) #Will create an error

Adding items to a list

Items can be added to a list using the .append() function.

myList = [] myList.append(1) #Add 1 to the list print(myList)

Removing items from a list

Items can be removed from a list using the .pop(n) function where n is the index to be removed. The pop function also returns the item that has been removed.

myList = [1,2,3] myList.pop(1) #Remove 2 from the list print(myList)

Operators and lists

Like strings and integers, lists can combined using the addition operator.

a = [1,2,3,4,5] b = [6,7,8,9,10] print(a + b) #Combine the lists

The len() function refers the number of items in the list.

a = [1,2,3,4,5] print(len(a)) #Get the length of the list
