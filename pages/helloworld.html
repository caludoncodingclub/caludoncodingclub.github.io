Hello, World!

Python is a simple language, using very straightforward syntax. The simplest directive in python is the print directive, which simply outputs a line.

To print a line in Python, just write:

print("This text will be printed")

Commenting

Commenting is used to clarify the code and highlight what is happening, thus isn't interpreted by Python

To write a comment start a line with the # symbol

#This is a comment #print("This text will not be printed")
