Rock, Paper, Scissors

In this project you are going to be making a Rock, Paper, Scissors game and play against the computer.

Step 1: The player's turn

We need to start by getting an input from the player. We also need to validate what the player is entering to make sure they answer Rock, Paper or Scissors.

    #If not, ask again

For example:

    #Check if valid input
    if player not in ["Rock","Paper","Scissors"]:
        print("Invalid Input, Check your spelling.")
        player = False

Step 7: Customise

The main game might be finished, but that doesn't mean the coding has to end. Here are a few things to try:

  • Change up scoring. Score 1 point for drawing and 2 points for winning.

  • Change up highscores. Try adding the top 3 scores to the game instead of a single highscore.

  • Feeling artsy. Add some ascii art to the game, for example using O, ___ and >8 for symbols.

  • Feeling creative. Create a new alternative to Rock, Paper, Scissors. For example: Fire, Logs, Water

  • Feeling technical. Add more options to the game, implement Rock, Paper, Scissors, Lizard, Spock.
