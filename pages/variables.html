Variables and Data Types

Variables are how programs store data to be used. Every variable has a unique name (called the identifier) and can be given a value.

Numbers

Python supports two types of numbers, integers and floating point numbers.

An integer is a strictly whole number. To define one in Python, use the following syntax:

myInteger = 7 print(myInteger)

A floating point number is a number with a decimal component (although it can still be a whole number).
To define one in Python, you may use one of the following syntax:

myFloat = 7.5 print(myFloat) myFloat = float(7) print(myFloat)

Strings

Strings are used to store text and are defined either with a single quote or a double quotes.

myString = "Hello" print(myString) myString = 'Hello' print(myString)

Booleans

Boolean variables refers to the constants True and False.

myBool = True print(myBool) myBool = False print(myBool)
